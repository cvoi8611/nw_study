# 네트워크 스터디 3주차 과제

### 리버스 프록시 구현
- 리눅스에서 구현하였으며, 방식은 EPOLL 방식을 사용함
- (서버recv -> 본인이 클라 send -> 본인클라 recv -> 서버 send)

- 클라이언트 => 리버스 프록시 서버 => 서버 => 리버스 프록시 서버 => 클라이언트 의 구조로 제작함

- 클라이언트에서 HelloWorld! 입력시 서버에서 버퍼값을 수정하여 Welcome!으로 응답함
